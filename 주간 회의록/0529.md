# 교수님 피드백 반영

## 압축 방식 변경

- zlib에서 ADPCM으로 변경
- ADPCM 압축과 패킷 이슈 시뮬레이터가 합쳐지면서 새로운 유형(진폭 변경)의 노이즈 발생

## 오토인코더 심화

- Variation Auto Encoder는 차후 Auto Encoder와의 성능 비교 예정

## 오토인코더 역할 분담

- 태원, 성윤 : 오토인코더 기반 지식 및 논문으로 예시 코드 실행 시도
- 유진, 민재 : 오토인코더 기초 공부 및 예시 코드 모색

# 현재까지의 에러 형태

- 2. 패킷 손실
